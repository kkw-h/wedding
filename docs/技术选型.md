# 技术选型

- 框架：React 18 + TypeScript
- 构建：Vite
- UI 组件库：Ant Design v5
- 状态管理：Redux Toolkit + RTK Query
- 路由：React Router v6
- HTTP：Axios
- 样式：CSS Modules + Sass
- Markdown：react-markdown + remark-gfm（禁用 HTML）
- 代码质量：ESLint + Prettier
 - 后端 API：Supabase（Postgres + RLS + REST/JS 客户端）
   - 前端环境变量：`VITE_SUPABASE_URL`、`VITE_SUPABASE_ANON_KEY`
   - JS 客户端：`@supabase/supabase-js`，统一在 `shared/api/supabase.ts` 创建客户端
