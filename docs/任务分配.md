# 任务分配计划

- 基础设施与框架搭建：初始化工程、布局、路由、状态管理、HTTP 客户端。
- 通用组件与工具：历史面板、Markdown 组件、分页 Hook、电话工具。
- 预案模块：列表/详情/编辑页与服务、类型定义。
- 流程模块：列表/编辑页与负责人选择器联动、时间排序。
- 场地模块：列表/详情/编辑页与附件缩略图。
- 人员模块：列表/详情/编辑页与权限控制。
- 预算模块：只读列表、筛选与搜索。

## 里程碑
-
- M1：工程与基础设施完成，项目可启动。
- M2：通用组件完成，预案与流程打通。
- M3：场地与人员完成，权限生效。
- M4：预算完成，历史记录联调。
- M5：文档完善与回归测试。
